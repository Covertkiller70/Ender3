# Summary

##### PiCase for Ender 3

This is a remix of the Ender 3 Pi Mount. I added a little Box in the back that will hold a LM2596S to get the 5V for the Raspberry Pi. Also added a toolbox / trashbox.

This will fit between the electronics box and the display on the Ender 3 (see images). 

The LM2596S can be found pretty cheap on Amazon, Aliexpress, Gearbest.
